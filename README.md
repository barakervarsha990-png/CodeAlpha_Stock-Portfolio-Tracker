## 📌 Project Overview

This project is a **console-based Stock Portfolio Tracker** developed using Python.  
It allows users to input stock names and quantities, and calculates the total investment, current value, and profit or loss based on predefined stock prices.

---

## 🎯 Objective

The main objective of this project is to implement **basic programming concepts** such as dictionaries, loops, conditional statements, and file handling in a practical financial application.

---

## 🚀 Features

- 📥 User input for stock names and quantities  
- 📊 Calculates total investment value  
- 📈 Displays current value and profit/loss  
- 💾 Save results to `.txt` and `.csv` files  
- ✅ Input validation for better user experience  
- 🎨 Clean and interactive console output

## ⚙️ Execution

To run this project, first ensure that Python 3 is installed on your system by checking the version using the command `python --version`. Next, clone the repository from GitHub using `git clone https://github.com/your-username/stock-portfolio-tracker.git` and navigate into the project folder using `cd stock-portfolio-tracker`. After that, run the program by executing `python main.py` in the terminal. The application will prompt you to enter stock names and their quantities, and once you finish entering data by typing `done`, it will display the total investment, current value, and profit or loss. You will also have an option to save the results into a `.txt` or `.csv` file for future reference.
